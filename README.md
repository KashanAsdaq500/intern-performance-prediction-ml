# Intern Performance Prediction Model

## Objective
To predict intern performance using machine learning based on:
- Task completion time
- Feedback ratings
- Attendance
- Tasks completed

## Model Used
- Random Forest Regressor

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib
- Google Colab

## Results
The model successfully predicts intern performance and identifies key factors affecting outcomes.

## Output
- Performance prediction
- Feature importance visualization
